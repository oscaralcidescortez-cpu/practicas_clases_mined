# Tarea 3 - Landing Page Agencia de Viajes

Sitio de una sola página para una agencia de viajes a Centroamérica con:

- Hero con imagen completa, título, subtítulo y CTA.
- Navbar fija con anclas y menú hamburguesa responsive.
- Sección "Nosotros".
- Sección "Servicios" con tarjetas en CSS Grid.
- Formulario validado con JavaScript.
- Footer con redes, año dinámico y créditos.

## Archivos

- `index.html`
- `styles.css`
- `script.js`

## Cómo usar

1. Abre `tarea3/index.html` en el navegador.
2. Completa el formulario y prueba la validación.
3. Haz clic en las tarjetas de servicio para ver detalles dinámicos.

## Preparación para despliegue

- Asegúrate de subir estos archivos a un repositorio Git.
- Para publicación gratuita, usa GitHub Pages, Netlify o Vercel.
- En GitHub Pages, activa la rama principal del repositorio y selecciona la carpeta `/tarea3` si es necesario.
- En Netlify o Vercel, crea un sitio conectado al repositorio y configura la carpeta raíz a `tarea3`.
