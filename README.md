## 📚 ¿Qué es este repositorio?
 
Este repositorio ha sido creado con fines **educativos y colaborativos** para los docentes participantes del **Bootcamp Tecnológico del MINEDUCYT (Ministerio de Educación, Ciencia y Tecnología)**. Aquí encontrarás todas las **prácticas, ejercicios y proyectos** desarrollados a lo largo del bootcamp, organizados por módulo y tecnología.
 
> 💡 *Este espacio es de aprendizaje compartido. ¡Explora, practica y comparte con tus compañeros docentes!*
 
---
 
## 👩‍🏫👨‍🏫 ¿Para quién es?
 
Este repositorio está dirigido a los **compañeros docentes** que participan en el bootcamp, como herramienta de apoyo para:
 
- 📖 Repasar los contenidos vistos en clases
- 💻 Acceder a las prácticas de cada módulo
- 🤝 Colaborar y aprender entre compañeros
- 🧪 Experimentar con el código de forma libre

## 🏁 ¿Cómo usar este repositorio?
 
### 1️⃣ Clona el repositorio
 
```bash
git clone https://github.com/tu-usuario/bootcamp-mineducyt.git
```
 
### 2️⃣ Navega a la práctica que necesitas
 
```bash
cd bootcamp-mineducyt/html/practica-01-estructura-basica
```
 
### 3️⃣ Abre el archivo en tu editor favorito
 
```bash
code .   # Si usas Visual Studio Code
```

## 🌟 Contribuir
 
¿Quieres aportar al repositorio? ¡Bienvenido!
 
1. Haz un **fork** del repositorio 🍴
2. Crea una rama con tu práctica: `git checkout -b practica/mi-ejercicio`
3. Realiza tus cambios y haz **commit**: `git commit -m "Agrego práctica de flexbox"`
4. Sube los cambios: `git push origin practica/mi-ejercicio`
5. Abre un **Pull Request** 🚀
