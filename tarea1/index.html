📖 Estructura del Curso

Módulo 1: Fundamentos

Duración: 2 semanas

Aprenderás los conceptos básicos de Python: variables, tipos de datos, operadores y estructuras de control.

  • Variables y tipos de datos (int, float, str, bool)
  • Operadores aritméticos y de comparación
  • Condicionales (if, elif, else)
  • Bucles (for, while)

Módulo 2: Estructuras de Datos

Duración: 2 semanas

Domina las estructuras de datos más importantes en Python y cómo manipularlas eficientemente.

  • Listas y operaciones
  • Tuplas e inmutabilidad
  • Diccionarios y acceso a datos
  • Conjuntos (Sets)

Módulo 3: Funciones y Módulos

Duración: 2 semanas

Crea código reutilizable y organizado mediante funciones y módulos personalizados.

  • Definición y llamada de funciones
  • Parámetros y argumentos
  • Valores de retorno
  • Importar y crear módulos

Módulo 4: Programación Orientada a Objetos

Duración: 3 semanas

Aprende el paradigma POO y cómo estructurar código con clases y objetos.

  • Clases y objetos
  • Atributos y métodos
  • Herencia y polimorfismo
  • Encapsulación

Módulo 5: Manejo de Archivos

Duración: 1 semana

Trabaja con archivos: lectura, escritura y manipulación de datos.

  • Lectura y escritura de archivos
  • Formatos CSV, JSON, XML
  • Manejo de excepciones
  • Context managers

Módulo 6: Proyectos Prácticos

Duración: 2 semanas

Aplica todo lo aprendido en proyectos reales y útiles.

  • Sistema de gestión de estudiantes
  • Calculadora avanzada
  • Web scraping básico
  • Automatización de tareas
